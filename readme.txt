モデルをGameに反映させるために
今までは、cmoファイルを読み込んでいましたが、今回からはtkmファイルを
読み込んでいくことになります。
そのため、teamG/tools/setup.batを初回のみ起動させておいてください。
また、今まで使っていたtkExporterではなく、teamG/tools/3dsMaxScripts/tkExporter.msを
使用してください。

スプライトを出すために
今回からpngやjpgではなくddsファイルを使用します。
ですので、画像ファイルをオンライン等でdds形式に変換してから使うようにしてください。
変換しただけだとddsファイルは使えないので、設定の変更をお願いします。。
この設定の変更はvs2019ではできないので、vs2017で行ってください。
次に、vs2017のソリューションにddsを追加し、項目の種類を変更してください。　
画像の初期設定では、
ddsのプロパティ/構成プロパティ/全般/項目の種類/Image
となっています。
ImageをImage Content Pipelineにしてください。
これで、正しく表示されます。

今までのようにソリューションにpngやjpg等の画像ファイルを追加する必要はありません。
